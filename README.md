## 🛠️ Tech Stack

## 📦 Modules
| Module | Features |
|--------|----------|
| Employee Management | Add, View, Search, Update, Delete, CSV Export |
| Department Management | Add, Update Manager, Delete |
| Salary Management | UPSERT Logic, Transactions, Salary Slip, Bonus Calc |
| Leave Management | Apply, Approve/Reject, Balance, Multi-Day Loop |
| Attendance Tracking | Mark, Summary, Duplicate Check |
| Reports & Analytics | Live Charts, Metric Cards, Dept Stats |

## ✨ Special Features
- 🔴 Live SQL Log (animated typewriter)
- 🌙 Dark Mode Toggle
- 📄 CSV Export all modules
- 🖨️ Salary Slip Print
- 🔄 ACID Transactions (COMMIT/ROLLBACK)
- 🔒 SQL Injection Prevention (Prepared Statements)
